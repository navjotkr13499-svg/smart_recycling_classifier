# Dataset Directory

## ⚠️ Important: Dataset files are NOT included in this repository

Due to size constraints, the image dataset is not stored in Git. Follow the instructions below to set up the data.

## 📥 How to Get the Data

### Option 1: Automatic Download (Recommended)
```bash
python src/data/download.py
```

### Option 2: Manual Download
1. Visit: https://www.kaggle.com/datasets/asdasdasasdas/garbage-classification
2. Download the dataset
3. Extract to `data/raw/` folder
4. Expected structure:
data/raw/
├── cardboard/
├── glass/
├── metal/
├── paper/
├── plastic/
└── trash/
### Option 3: TrashNet Dataset
1. Visit: https://github.com/garythung/trashnet
2. Download and extract
3. Place in `data/raw/` folder

## 📁 Directory Structure
data/
├── raw/              # Original images (download required)
├── processed/        # Processed train/val/test split (generated by scripts)
├── sample_images/    # Small demo samples (included in repo)
└── README.md         # This file
## 🔄 After Downloading Data

Run the preprocessing script to split data:
```bash
python src/data/preprocess.py
```

This will create:
- `data/processed/train/` - Training set (70%)
- `data/processed/val/` - Validation set (15%)
- `data/processed/test/` - Test set (15%)
